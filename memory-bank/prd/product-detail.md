# PRD — 상품 상세 페이지

> 마지막 갱신: 2026-05-08

## 1. 개요

### 목적

사용자가 특정 TCG 카드의 상세 정보, 현재 가격, 과거 가격 변동 추이를 확인하여 구매/판매 결정을 내릴 수 있도록 한다.

### 대상 사용자

- 특정 카드 구매를 고려하는 사용자.
- 보유 카드 판매 시점을 판단하려는 사용자.
- 카드 가격 변동을 추적하는 컬렉터.

### 진입 경로

- 검색 결과 카드 클릭.
- 카테고리 카드 클릭.
- 관심 카드 목록.
- 외부 공유 링크.

### 다음 이동 경로

- 이전 검색 결과.
- 동일 세트/카테고리 페이지.
- 외부 거래처 링크.

## 2. 목표와 기능 정의

### 목표

- 카드의 현재 시장 가격을 명확하게 제공한다.
- 가격 변동 추이를 시각적으로 보여준다.
- 사용자가 거래 판단에 필요한 정보를 한 화면에서 확인할 수 있다.

### 핵심 기능

- 카드 기본 정보.
- 평균가/최고가/최저가.
- 가격 변동 차트.
- 데이터 출처 및 업데이트 시간.

## 3. 사용자 스토리

- As a 카드를 구매하려는 사용자로서  
  I want to 해당 카드의 평균가와 최저가를 확인하고 싶다  
  So that 비싸게 구매하는 것을 피할 수 있다.

- As a 카드를 판매하려는 사용자로서  
  I want to 최고가와 가격 상승 추이를 확인하고 싶다  
  So that 적절한 판매 시점을 판단할 수 있다.

- As a 컬렉터로서  
  I want to 관심 카드를 저장하고 싶다  
  So that 이후 가격 변동을 계속 추적할 수 있다.

## 4. 기능 요구 사항

### P0: 필수 요구 사항

- 카드 이미지를 노출한다.
- 카드명, 게임명, 세트명, 레어도를 노출한다.
- 평균 거래가, 최저가, 최고가를 노출한다.
- 최근 가격 업데이트 시간을 노출한다.
- 가격 변동 차트를 제공한다.
- 가격 데이터 출처를 표시한다.
- 이전 페이지로 돌아가기 동선을 제공한다.
- 존재하지 않는 카드 접근 시 404 또는 에러 상태를 제공한다.

### P1: 권장 요구 사항

- 관심 카드 등록.
- 가격 알림 설정.
- 외부 거래처 링크.
- 기간별 차트 필터: 7일, 30일, 90일, 1년.
- 동일 세트의 관련 카드 추천.
- 가격 상승/하락률 표시.
- 보유 수량 입력을 통한 컬렉션 가치 계산.

## 5. 상태 정의

- 기본 상태: 카드 상세 정보와 가격 차트 노출.
- 로딩 상태: 상세 정보/차트 로딩.
- 빈 상태: 가격 데이터가 부족한 경우 안내.
- 에러 상태: 카드 정보 조회 실패 또는 가격 데이터 조회 실패.
- 인증 상태: 관심 카드 등록은 로그인 사용자만 가능.

## 6. 필요한 데이터

- cardId: 카드 고유 ID.
- cardName: 카드명.
- gameTitle: TCG 종류.
- setName: 세트/확장팩명.
- rarity: 레어도.
- imageUrl: 카드 이미지.
- avgPrice: 평균 거래가.
- minPrice: 최저가.
- maxPrice: 최고가.
- priceHistory: 날짜별 가격 데이터.
- sources: 가격 데이터 출처.
- lastUpdatedAt: 마지막 업데이트 시간.
