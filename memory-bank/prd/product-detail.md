# PRD — 상품 상세 페이지

> 마지막 갱신: 2026-07-07

## 1. 개요

### 목적

사용자가 특정 TCG 카드의 상세 정보, 현재 가격, 과거 가격 변동 추이를 확인하여 구매/판매 결정을 내릴 수 있도록 한다.

### 대상 사용자

- 특정 카드 구매를 고려하는 사용자.
- 보유 카드 판매 시점을 판단하려는 사용자.
- 카드 가격 변동을 추적하는 컬렉터.

### 진입 경로

- 검색 결과 카드 클릭.
- 카테고리 카드 클릭.
- 관심 카드 목록.
- 외부 공유 링크.

### 다음 이동 경로

- 이전 검색 결과.
- 동일 세트/카테고리 페이지.
- 외부 거래처 링크.

## 2. 목표와 기능 정의

### 목표

- 카드의 현재 시장 가격을 명확하게 제공한다.
- 가격 변동 추이를 시각적으로 보여준다.
- 사용자가 거래 판단에 필요한 정보를 한 화면에서 확인할 수 있다.

### 핵심 기능

- 카드 기본 정보.
- 평균/최고/최저 판매 호가.
- 가격 변동 차트.
- 데이터 출처 및 업데이트 시간.

## 3. 사용자 스토리

- As a 카드를 구매하려는 사용자로서  
  I want to 해당 카드의 평균가와 최저가를 확인하고 싶다  
  So that 비싸게 구매하는 것을 피할 수 있다.

- As a 카드를 판매하려는 사용자로서  
  I want to 최고가와 가격 상승 추이를 확인하고 싶다  
  So that 적절한 판매 시점을 판단할 수 있다.

- As a 컬렉터로서  
  I want to 관심 카드를 저장하고 싶다  
  So that 이후 가격 변동을 계속 추적할 수 있다.

## 4. 기능 요구 사항

### P0: 필수 요구 사항

- 카드 이미지를 노출한다.
- 상품 상세는 기본 한국판(`ko/KR`)으로 열고, 같은 카드에 일본판(`ja/JP`) 또는 미국판/북미판(`en/NA`) printing이 있으면 사용자가 판본을 선택할 수 있다.
- 판본 선택 시 카드 이미지, 가격 요약, 가격 변동 차트는 선택한 `cardPrintingId` 기준으로 변경한다.
- 카드명, 게임명, 세트명, 레어도를 노출한다.
- 대표 가격은 판매중 호가(`asking`) snapshot만 사용하며 `평균 판매 호가`로 표시한다.
- 판매 호가의 최저가와 최고가를 노출한다.
- 판매 호가 이력이 없으면 임의 가격을 만들지 않고 `시세 정보 없음`을 표시한다.
- 과거 판매 호가만 있으면 마지막 값을 유지하고 `마지막 수집 N일 전 · 현재 매물 여부 미확인`을 표시한다.
- 검증된 개별 매물 링크가 없으면 실제 snapshot 출처 링크를 표시하고, 출처 링크도 없으면 eBay 검색 결과로 이동한다.
- 판매 완료 데이터는 대표 가격 계산에서 제외하고 차트 참고 데이터로만 유지한다.
- 가격 기준 시장(`KR`, `JP`, `NA`)과 언어판/지역판/호일/상태/그레이딩 구분을 섞지 않는다.
- 최근 가격 업데이트 시간을 노출한다.
- 가격 변동 차트를 제공한다.
- 가격 데이터 출처를 표시한다.
- 가격 산정 표본 수를 표시한다.
- 이전 페이지로 돌아가기 동선을 제공한다.
- 존재하지 않는 카드 접근 시 404 또는 에러 상태를 제공한다.

### P1: 권장 요구 사항

- 관심 카드 등록.
- 가격 알림 설정.
- 기간별 차트 필터: 7일, 30일, 90일, 1년.
- 동일 세트의 관련 카드 추천.
- 가격 상승/하락률 표시.
- 보유 수량 입력을 통한 컬렉션 가치 계산.

## 5. 상태 정의

- 기본 상태: 카드 상세 정보와 가격 차트 노출.
- 로딩 상태: 상세 정보/차트 로딩.
- 빈 상태: 가격 데이터가 부족한 경우 안내.
- 에러 상태: 카드 정보 조회 실패 또는 가격 데이터 조회 실패.
- 인증 상태: 관심 카드 등록은 로그인 사용자만 가능.

## 6. 필요한 데이터

- cardId: 카드 고유 ID.
- cardName: 카드명.
- gameTitle: TCG 종류.
- setName: 세트/확장팩명.
- rarity: 레어도.
- imageUrl: 카드 이미지.
- avgPrice: 평균 판매 호가. 판매 호가 snapshot이 없으면 `null`.
- minPrice: 최저 판매 호가.
- maxPrice: 최고 판매 호가.
- market: 가격 시장(`KR`, `JP`, `NA`).
- cardPrintingId: 언어판/지역판/세트/번호/호일 단위 식별자.
- selectedEdition: 선택된 판본(`kr`, `jp`, `na`). 기본값은 `kr`.
- availableEditions: 선택 가능한 판본 목록과 각 판본의 printing 식별자.
- conditionLabel: 가격 산정 상태 라벨.
- gradeCompany, gradeValue: 그레이딩 가격일 때 등급 식별자.
- sampleCount: 가격 산정 표본 수.
- priceHistory: 날짜별 가격 데이터.
- sources: 가격 데이터 출처.
- lastUpdatedAt: 마지막 업데이트 시간.
