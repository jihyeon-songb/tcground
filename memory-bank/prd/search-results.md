# PRD — 검색 결과 페이지

> 마지막 갱신: 2026-05-08

## 1. 개요

### 목적

사용자가 입력한 검색어와 관련된 TCG 카드 목록을 확인하고, 가격 정보를 비교한 뒤 상품 상세 페이지로 이동할 수 있도록 한다.

### 대상 사용자

- 특정 카드 시세를 확인하려는 사용자.
- 유사 카드 가격을 비교하려는 사용자.
- 구매 또는 판매 전 기준 가격을 알고 싶은 사용자.

### 진입 경로

- 홈페이지 검색창.
- 헤더 검색창.
- 검색어 자동완성 선택.

### 다음 이동 경로

- 상품 상세 페이지.
- 카테고리 페이지.
- 재검색 결과 페이지.

## 2. 목표와 기능 정의

### 목표

- 검색어와 관련된 카드 목록을 정확하게 제공한다.
- 사용자가 카드별 평균가, 최고가, 최저가를 빠르게 비교할 수 있다.
- 결과가 없거나 오류가 발생한 상황에서도 다음 행동을 안내한다.

### 핵심 기능

- 검색 결과 목록.
- 가격 요약 정보.
- 정렬/필터 기반 탐색.

## 3. 사용자 스토리

- As a 카드를 구매하려는 사용자로서  
  I want to 검색 결과에서 카드별 가격을 비교하고 싶다  
  So that 적정 구매 가격을 판단할 수 있다.

- As a 카드를 판매하려는 사용자로서  
  I want to 현재 평균 거래가와 최고가를 확인하고 싶다  
  So that 판매 가격을 합리적으로 정할 수 있다.

## 4. 기능 요구 사항

### P0: 필수 요구 사항

- 검색어 기준 카드 목록을 노출한다.
- 검색 결과 개수를 노출한다.
- 카드 이미지, 카드명, 게임명, 세트명을 노출한다.
- 평균가, 최저가, 최고가를 노출한다.
- 최근 업데이트 시간을 노출한다.
- 카드 클릭 시 상품 상세 페이지로 이동한다.
- 검색 결과 없음 화면을 제공한다.
- 로딩/에러 상태를 제공한다.

### P1: 권장 요구 사항

- 가격순 정렬.
- 최근 업데이트순 정렬.
- 인기순 정렬.
- TCG 종류 필터.
- 세트/확장팩 필터.
- 레어도 필터.
- 페이지네이션 또는 무한 스크롤.

## 5. 상태 정의

- 기본 상태: 검색 결과 목록 노출.
- 로딩 상태: 스켈레톤 카드 목록 표시.
- 빈 상태: "검색 결과가 없습니다" 안내.
- 에러 상태: 검색 실패 시 재시도 버튼 제공.

## 6. 필요한 데이터

- cardId: 카드 고유 ID.
- cardName: 카드명.
- gameTitle: TCG 종류.
- setName: 세트/확장팩명.
- rarity: 레어도.
- thumbnailUrl: 카드 이미지.
- avgPrice: 평균 거래가.
- minPrice: 최저가.
- maxPrice: 최고가.
- lastUpdatedAt: 가격 업데이트 시간.
