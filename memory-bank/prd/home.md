# PRD — 홈페이지

> 마지막 갱신: 2026-05-08

## 1. 개요

### 목적

사용자가 서비스에 진입한 뒤 카드명을 검색하거나 주요 카테고리를 통해 카드 가격 정보를 탐색할 수 있도록 한다.

### 대상 사용자

- 특정 카드 가격을 알고 싶은 사용자.
- 인기 카드나 카테고리별 카드를 탐색하려는 사용자.
- TCG 시장 흐름을 빠르게 파악하려는 사용자.

### 진입 경로

- 직접 URL 접속.
- 로그인 성공 후 이동.
- 외부 공유 링크에서 홈으로 이동.

### 다음 이동 경로

- 검색 결과 페이지.
- 카테고리 대분류/소분류 페이지.
- 상품 상세 페이지.

## 2. 목표와 기능 정의

### 목표

- 사용자가 원하는 TCG 카드를 빠르게 검색할 수 있다.
- 주요 TCG 카테고리와 인기 카드에 접근할 수 있다.
- 첫 화면에서 서비스의 핵심 기능인 가격 탐색을 바로 사용할 수 있다.

### 핵심 기능

- 카드 검색.
- 주요 카테고리 진입.
- 인기 카드 또는 급등 카드 노출.

## 3. 사용자 스토리

- As a 카드 구매를 고려하는 사용자로서  
  I want to 카드명을 검색하고 싶다  
  So that 현재 적정 구매 가격을 확인할 수 있다.

- As a TCG 컬렉터로서  
  I want to 인기 카드나 급등 카드를 확인하고 싶다  
  So that 시장 흐름을 빠르게 파악할 수 있다.

## 4. 기능 요구 사항

### P0: 필수 요구 사항

- 검색 입력창을 제공한다.
- 검색 버튼 또는 엔터 검색을 지원한다.
- 카드명 검색 시 검색 결과 페이지로 이동한다.
- 빈 검색어 입력 시 검색 실행을 제한한다.
- 주요 TCG 카테고리를 노출한다.
- 인기 카드 영역을 제공한다.

### P1: 권장 요구 사항

- 검색어 자동완성.
- 사진으로 검색.
- 최근 검색어.
- 인기 검색어.
- 가격 급등 카드 영역.

## 5. 상태 정의

- 기본 상태: 검색창, 주요 카테고리, 인기 카드 노출.
- 로딩 상태: 인기 카드/카테고리 데이터 로딩.
- 빈 상태: 인기 카드 데이터가 없을 경우 기본 안내 문구.
- 에러 상태: 데이터 로딩 실패 시 재시도 제공.

## 6. 필요한 데이터

- categoryId: 카테고리 고유 ID.
- categoryName: 카테고리 이름.
- cardId: 카드 고유 ID.
- cardName: 카드명.
- thumbnailUrl: 카드 이미지.
- avgPrice: 평균 거래가.
- lastUpdatedAt: 가격 업데이트 시간.
