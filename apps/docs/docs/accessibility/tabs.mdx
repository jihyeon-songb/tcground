# Tabs — roving tabindex와 방향키 이동

탭의 핵심 접근성 패턴은 **roving tabindex** 입니다. 탭 목록 전체가 `Tab` 정지점 하나만 차지하고,
그 안에서는 **방향키**로 탭을 이동·활성화합니다. `@tcground/headless` 의 `Tabs` 는
`tablist` / `tab` / `tabpanel` role 과 `aria-selected`, `aria-controls` 연결을 기본 제공합니다.

## 직접 확인

- ✅ 버전: `Tab` 으로 탭 목록에 진입하면 활성 탭에 focus → 좌우 방향키로 탭 이동 시 자동 활성화.
- ❌ 버전: `Tab` 으로 탭에 focus 가 가지 않고, 방향키도 동작하지 않습니다. 마우스로만 전환할 수 있습니다.

## headless가 보장하는 것

- `TabsList` 는 `role="tablist"` 를 가지며 `aria-label` 로 목적을 알릴 수 있습니다.
- `TabsTrigger` 는 `role="tab"`, `aria-selected`, `aria-controls`, roving `tabIndex`(활성 0 / 비활성 -1)를 가집니다.
- `TabsContent` 는 `role="tabpanel"`, `aria-labelledby` 로 트리거와 연결됩니다.
- `ArrowLeft` / `ArrowRight` 로 trigger 사이를 이동하면 해당 탭이 활성화됩니다.
