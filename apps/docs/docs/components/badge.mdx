# Badge

상태, 필터 값, 카테고리, 수량처럼 짧은 메타 정보를 텍스트 흐름 안에서 표시하는 컴포넌트입니다.
`Badge`는 `span`을 기본 렌더링하지만 `asChild`로 링크나 다른 요소에 스타일을 위임할 수 있습니다.

</Badge>`}
>
  <WithIconExample />
</ComponentPreview>

## 설치

```bash
pnpm add @tcground/ui
```

```ts
import '@tcground/ui/theme.css';
```

## 사용법

```tsx
import { Badge } from '@tcground/ui';

export function Example() {
  return <Badge variant='secondary'>검증 완료</Badge>;
}
```

`Badge`는 기본적으로 native `<span>` props를 받습니다. `asChild`를 사용하면 자식 요소의 props와 semantics를 따릅니다.

## 접근성

- Badge는 보통 보조 메타 정보입니다. 단독으로 중요한 상태를 전달해야 하면 주변 텍스트에도 같은 의미를 적습니다.
- 링크 Badge는 `asChild`와 `<a>`를 함께 사용해 키보드 탐색과 링크 semantics를 유지합니다.
- 아이콘은 텍스트와 같은 의미를 반복하면 `aria-hidden="true"`로 숨깁니다.
