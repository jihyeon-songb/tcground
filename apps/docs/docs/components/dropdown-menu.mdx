# Dropdown Menu

Trigger 를 누르면 메뉴 항목을 나열하는 floating 컴포넌트입니다.
Radix Dropdown Menu primitive 위에 TCGround 스타일을 얹은 styled UI 컴포넌트라 `theme.css`만 불러와도 content, item, disabled, highlighted 상태의 기본 스타일이 포함됩니다.

```bash
pnpm add @tcground/ui
```

```ts
import '@tcground/ui/theme.css';
```

## 사용법

`DropdownMenu` 안에 `DropdownMenuTrigger`, `DropdownMenuContent`, 그리고 한 개 이상의 `DropdownMenuItem` 을 둡니다.
Trigger 는 기본 버튼을 렌더할 수도 있지만, 앱과 문서에서는 `asChild`로 `Button`에 동작을 위임하는 패턴을 권장합니다.

```tsx
import {
  Button,
  DropdownMenu,
  DropdownMenuContent,
  DropdownMenuItem,
  DropdownMenuLabel,
  DropdownMenuSeparator,
  DropdownMenuShortcut,
  DropdownMenuTrigger,
} from '@tcground/ui';

export function Example() {
  return (
    <DropdownMenu>
      <DropdownMenuTrigger asChild>
        <Button variant='outline'>옵션</Button>
      </DropdownMenuTrigger>
      <DropdownMenuContent>
        <DropdownMenuLabel>카드 작업</DropdownMenuLabel>
        <DropdownMenuItem onClick={() => console.log('상세 보기')}>
          상세 보기 <DropdownMenuShortcut>Enter</DropdownMenuShortcut>
        </DropdownMenuItem>
        <DropdownMenuSeparator />
        <DropdownMenuItem variant='destructive'>관심 목록에서 제거</DropdownMenuItem>
      </DropdownMenuContent>
    </DropdownMenu>
  );
}
```

### DropdownMenuTrigger

Radix Trigger props를 받습니다. 기본 렌더는 native `<button>` 이고, `asChild` 사용 시 자식 요소에 ARIA와 이벤트가 머지됩니다.

- `aria-haspopup="menu"`, `aria-expanded` 자동 적용
- `ArrowDown` / `Enter` / `Space` 로 메뉴 열기
- click 시 toggle 동작

### DropdownMenuContent

Radix Content props 를 받습니다. `document.body` 로 portal 되며, placement는 Radix floating 로직을 따릅니다.

- `role="menu"`, `aria-labelledby={triggerId}` 자동 적용
- `ArrowDown` / `ArrowUp` 으로 active item 이동
- `Escape` 로 메뉴 닫기

## 접근성

- Trigger 는 `aria-haspopup="menu"` 와 `aria-expanded` 를 가집니다.
- Content 는 `role="menu"` 와 `aria-labelledby={triggerId}` 로 trigger 에 연결됩니다.
- Item 은 `role="menuitem"` 을 갖고, active 인 항목만 `tabIndex={0}` 으로 focus 됩니다.
- 키보드 지원: `ArrowDown` / `ArrowUp` 으로 항목 이동, `Enter` / `Space` 또는 클릭으로 선택, `Escape` 로 닫기.
- mouse hover 시 해당 항목이 active 가 되어 키보드 focus 와 일치합니다.
