# Dropdown Menu

trigger 를 누르면 메뉴 항목을 나열하는 floating 컴포넌트입니다.
방향키로 항목 사이를 이동하고, ESC 로 닫고, mouse hover 시 active 항목이 바뀝니다. 항목을 선택하면 메뉴가 자동으로 닫힙니다.

## 설치

```bash
pnpm add @tcground/headless-ui
```

```ts
import '@tcground/headless-ui/theme.css';
```

## 사용법

`DropdownMenu.Root` 안에 `Trigger`, `Content`, 그리고 한 개 이상의 `Item` 을 둡니다. Root 는 trigger 와 content 의 위치 관계를 잡기 위해 `position: relative` 컨테이너를 렌더합니다.

```tsx
import { DropdownMenu } from '@tcground/headless-ui';

export function Example() {
  return (
    <DropdownMenu.Root>
      <DropdownMenu.Trigger>메뉴</DropdownMenu.Trigger>
      <DropdownMenu.Content>
        <DropdownMenu.Item onClick={() => console.log('첫 번째')}>첫 번째</DropdownMenu.Item>
        <DropdownMenu.Item onClick={() => console.log('두 번째')}>두 번째</DropdownMenu.Item>
      </DropdownMenu.Content>
    </DropdownMenu.Root>
  );
}
```

### DropdownMenu.Trigger

native `<button>` 의 모든 props 를 받습니다. 컴포넌트가 추가하는 ARIA / 키보드 동작:

- `aria-haspopup="menu"`, `aria-expanded` 자동 적용
- `ArrowDown` / `Enter` / `Space` 로 메뉴 열기
- click 시 toggle 동작

### DropdownMenu.Content

native `<div>` 의 모든 props 를 받습니다. 컴포넌트가 추가하는 동작:

- `role="menu"`, `aria-labelledby={triggerId}` 자동 적용
- `ArrowDown` / `ArrowUp` 으로 active item 이동
- `Escape` 로 메뉴 닫기
- `context.open === false` 이면 아예 렌더되지 않습니다.

native `<button>` 의 props 를 받습니다. `role="menuitem"` 과 `tabIndex` (활성: 0, 비활성: -1) 가 자동 적용되고, click 시 메뉴가 닫힙니다.

## 접근성

- Trigger 는 `aria-haspopup="menu"` 와 `aria-expanded` 를 가집니다.
- Content 는 `role="menu"` 와 `aria-labelledby={triggerId}` 로 trigger 에 연결됩니다.
- Item 은 `role="menuitem"` 을 갖고, active 인 항목만 `tabIndex={0}` 으로 focus 됩니다.
- 키보드 지원: `ArrowDown` / `ArrowUp` 으로 항목 이동, `Enter` / `Space` 또는 클릭으로 선택, `Escape` 로 닫기.
- mouse hover 시 해당 항목이 active 가 되어 키보드 focus 와 일치합니다.
