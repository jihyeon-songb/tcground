# Button

폼 제출, 다이얼로그 열기, 라우팅처럼 사용자의 액션을 트리거하는 기본 인터랙티브 요소입니다.
`@tcground/ui`는 styled UI library이므로 `<Button>Save changes</Button>`만으로도 깔끔한 기본 버튼을 렌더링합니다.
기본은 native `<button>`을 렌더링하고, 링크나 라우터 컴포넌트에 동작을 위임해야 할 때만 `asChild`를 사용합니다.

  <DefaultExample />
</ComponentPreview>

## 설치

```bash
pnpm add @tcground/ui
```

TCG 토큰(`--primary`, `--tcg-red`, `--tcg-red-dark`, `--ring` 등)을 활성화하려면 글로벌 CSS 진입점에서 한 번만 import 합니다.

```ts
import '@tcground/ui/theme.css';
```

## 사용법

```tsx
import { Button } from '@tcground/ui';

export function Example() {
  return <Button onClick={() => console.log('clicked')}>저장</Button>;
}
```

기본 스타일은 primary solid 버튼이며, `className`으로 필요한 화면에 맞게 덮어쓸 수 있습니다.
`type` prop 의 기본값은 `'button'` 이라, form 내부에서 의도치 않은 submit이 발생하지 않습니다.
실제로 submit이 필요하면 `type="submit"` 을 명시합니다.

### Disabled

`disabled` prop은 native `<button>`에서는 DOM `disabled` 속성으로, `asChild` 위임 시에는 `aria-disabled="true"` + `tabIndex={-1}` 로 표현됩니다. 두 경우 모두 `onClick` 호출이 차단됩니다.

</ComponentPreview>

## 접근성

- 기본 `type="button"` 으로 form 내부의 의도치 않은 submit 을 방지합니다.
- `disabled` 상태는 native 모드에서는 DOM `disabled` 로, `asChild` 모드에서는 `aria-disabled="true"` + `tabIndex={-1}` 로 표현됩니다.
- focus 링은 `--ring` 토큰을 사용하며, `:focus-visible` 에서만 노출됩니다.
- `asChild` 로 위임할 때 자식 요소의 기존 `onClick`, `ref`, `tabIndex` 는 손실 없이 머지됩니다.
