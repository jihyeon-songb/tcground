# Radio Group

상호 배타적인 선택지 중 하나의 값을 고르는 form control입니다.
`RadioGroup`은 그룹 상태와 키보드 이동을 관리하고, `RadioGroupItem`은 각 선택지를 렌더합니다.

<ComponentPreview
  code={`<RadioGroup defaultValue="kr">
  <Label><RadioGroupItem value="kr" /> 한국판</Label>
  <Label><RadioGroupItem value="jp" /> 일본판</Label>
  <Label><RadioGroupItem value="en" /> 영문판</Label>
</RadioGroup>`}
>
  <DefaultExample />
</ComponentPreview>

## 설치

```bash
pnpm add @tcground/ui
```

```ts
import '@tcground/ui/theme.css';
```

## 사용법

```tsx
import { Label, RadioGroup, RadioGroupItem } from '@tcground/ui';

export function Example() {
  return (
    <RadioGroup defaultValue='kr'>
      <Label>
        <RadioGroupItem value='kr' /> 한국판
      </Label>
      <Label>
        <RadioGroupItem value='jp' /> 일본판
      </Label>
    </RadioGroup>
  );
}
```

## 접근성

- 각 `RadioGroupItem`에는 보이는 라벨을 제공합니다.
- 방향키로 그룹 안의 선택지를 이동하고, `Space` 키로 선택할 수 있습니다.
- 한 그룹 안에서는 하나의 값만 선택됩니다.
- controlled 사용 시 `value`와 `onValueChange`를 함께 사용합니다.
