# Select

제한된 옵션 목록에서 하나의 값을 고르는 컴포넌트입니다.
Radix Select primitive 기반이며, trigger, content, group, item, separator를 조합해 사용합니다.

<ComponentPreview
  code={`<Select>
  <SelectTrigger>
    <SelectValue placeholder="카테고리 선택" />
  </SelectTrigger>
  <SelectContent>
    <SelectItem value="pokemon">포켓몬</SelectItem>
    <SelectItem value="magic">매직: 더 개더링</SelectItem>
  </SelectContent>
</Select>`}
>
  <BasicExample />
</ComponentPreview>

## 설치

```bash
pnpm add @tcground/ui
```

```ts
import '@tcground/ui/theme.css';
```

## 사용법

```tsx
import { Select, SelectContent, SelectItem, SelectTrigger, SelectValue } from '@tcground/ui';

export function Example() {
  return (
    <Select>
      <SelectTrigger>
        <SelectValue placeholder='카테고리 선택' />
      </SelectTrigger>
      <SelectContent>
        <SelectItem value='pokemon'>포켓몬</SelectItem>
        <SelectItem value='magic'>매직: 더 개더링</SelectItem>
      </SelectContent>
    </Select>
  );
}
```

## 접근성

- trigger는 현재 선택값과 expanded 상태를 보조 기기에 전달합니다.
- `ArrowDown` / `ArrowUp`으로 항목을 이동하고, `Enter` 또는 `Space`로 선택합니다.
- `SelectValue`의 placeholder는 값이 없을 때만 표시됩니다.
- 옵션이 많으면 `SelectGroup`, `SelectLabel`, `SelectSeparator`로 구획을 나눕니다.
