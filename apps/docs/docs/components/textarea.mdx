# Textarea

여러 줄 메모나 설명을 입력받는 form control입니다.
`Textarea`는 native `<textarea>` props를 그대로 받으며, `Input`과 같은 token 기반 border, focus, disabled, invalid 상태를 사용합니다.

  <DefaultExample />
</ComponentPreview>

## 설치

```bash
pnpm add @tcground/ui
```

```ts
import '@tcground/ui/theme.css';
```

## 사용법

```tsx
import { Label, Textarea } from '@tcground/ui';

export function Example() {
  return (
    <div>
      <Label htmlFor='card-note'>카드 메모</Label>
      <Textarea id='card-note' placeholder='상태와 보관 위치를 기록하세요' />
    </div>
  );
}
```

## 접근성

- `Label`의 `htmlFor`와 `Textarea`의 `id`를 연결합니다.
- 긴 설명이나 에러 문구는 `aria-describedby`로 연결할 수 있습니다.
- placeholder는 라벨을 대체하지 않습니다.
- `disabled`는 입력, focus, form 제출 값을 모두 비활성화합니다.
