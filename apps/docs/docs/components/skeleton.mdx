# Skeleton

데이터 로딩 중 실제 콘텐츠의 대략적인 형태를 먼저 보여주는 placeholder입니다.
`Skeleton`은 크기를 직접 지정해 쓰는 작은 primitive이며, 주변 layout의 안정적인 폭과 높이가 중요합니다.

  code={`<div>

  <Skeleton style={{ height: '1rem', width: '75%' }} />
  <Skeleton style={{ height: '1rem', width: '50%' }} />
</div>`}
>
  <BlockExample />
</ComponentPreview>

## 설치

```bash
pnpm add @tcground/ui
```

```ts
import '@tcground/ui/theme.css';
```

## 사용법

```tsx
import { Skeleton } from '@tcground/ui';

export function Example() {
  return <Skeleton style={{ height: '1rem', width: '12rem' }} />;
}
```

## 접근성

- Skeleton 자체는 의미 있는 콘텐츠가 아닙니다. 필요한 경우 주변 영역에 `aria-busy="true"`를 설정합니다.
- 로딩이 길어질 수 있으면 상태 텍스트를 시각적으로 숨긴 형태로 함께 제공합니다.
- 실제 콘텐츠와 비슷한 크기로 만들어 layout shift를 줄입니다.
- 깜박임이 강한 커스텀 애니메이션은 피합니다.
