# Command

카드 검색, 빠른 이동, 설정 명령처럼 keyboard 중심 탐색이 필요한 목록 UI에 사용합니다.
`cmdk` 기반이며, standalone command list와 `CommandDialog` command palette를 모두 지원합니다.

  <CommandList>

    <CommandGroup heading="제안">
      <CommandItem>가격 알림 설정</CommandItem>

    </CommandGroup>

    <CommandGroup heading="설정">
      <CommandItem>
        환경설정
        <CommandShortcut>⌘,</CommandShortcut>
      </CommandItem>
    </CommandGroup>
  </CommandList>
</Command>`}
>
  <DefaultExample />
</ComponentPreview>

## 설치

```bash
pnpm add @tcground/ui
```

```ts
import '@tcground/ui/theme.css';
```

## 사용법

```tsx
import {
  Command,
  CommandEmpty,
  CommandGroup,
  CommandInput,
  CommandItem,
  CommandList,
  CommandShortcut,
} from '@tcground/ui';

export function Example() {
  return (
    <Command>
      <CommandInput placeholder='카드 검색...' />
      <CommandList>
        <CommandEmpty>결과가 없습니다.</CommandEmpty>
        <CommandGroup heading='카드'>
          <CommandItem>리자몽 ex 151 SAR</CommandItem>
          <CommandItem>
            피카츄 ex SAR
            <CommandShortcut>⌘P</CommandShortcut>
          </CommandItem>
        </CommandGroup>
      </CommandList>
    </Command>
  );
}
```

## 접근성

- CommandInput은 listbox 검색 입력으로 동작하며, 항목 탐색은 `cmdk` keyboard behavior를 따릅니다.
- 결과가 없을 때는 `CommandEmpty`로 명시적인 빈 상태를 제공합니다.
- 그룹이 많으면 `CommandGroup`과 `CommandSeparator`로 의미 단위를 나눕니다.
- Dialog 형태로 사용할 때는 `CommandDialog`의 `title`과 `description`을 실제 목적에 맞게 지정합니다.
