# Checkbox

독립적인 boolean 값을 입력받는 체크박스입니다.
checkbox role, `aria-checked`(indeterminate 포함), Space 토글 같은 동작은 unstyled 패키지 `@tcground/headless`가 담당하고, `@tcground/ui`는 그 위에 checked, disabled, invalid 상태를 token으로 표현한 styled 레이어입니다.

</Label>`}
>
  <WithLabelExample />
</ComponentPreview>

## 설치

```bash
pnpm add @tcground/ui
```

```ts
import '@tcground/ui/theme.css';
```

## 사용법

```tsx
import { Checkbox, Label } from '@tcground/ui';

export function Example() {
  return (
    <Label>
      <Checkbox defaultChecked /> 가격 알림 받기
    </Label>
  );
}
```

## 접근성

- label 텍스트와 함께 사용해 클릭 영역과 이름을 제공합니다.
- `Space` 키로 checked 상태를 토글할 수 있습니다.
- controlled 사용 시 `checked`와 `onCheckedChange`를 짝지어 화면과 상태가 어긋나지 않게 합니다.
- 필수 약관처럼 반드시 체크해야 하는 항목은 서버 검증도 함께 수행합니다.
