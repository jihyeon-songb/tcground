# Card

관련된 정보와 액션을 하나의 표면으로 묶는 layout primitive입니다.
`Card`는 header, content, footer, action 영역을 조합해 목록 항목, 요약 패널, 설정 패널에 사용할 수 있습니다.

## 설치

```bash
pnpm add @tcground/ui
```

```ts
import '@tcground/ui/theme.css';
```

## 사용법

```tsx
import {
  Card,
  CardContent,
  CardDescription,
  CardFooter,
  CardHeader,
  CardTitle,
} from '@tcground/ui';

export function Example() {
  return (
    <Card>
      <CardHeader>
        <CardTitle>리자몽 ex 151 SAR</CardTitle>
        <CardDescription>한국판 / SV 강화확장팩</CardDescription>
      </CardHeader>
      <CardContent>최근 30일 평균 거래가가 12% 상승했습니다.</CardContent>
      <CardFooter>상세보기</CardFooter>
    </Card>
  );
}
```

### CardHeader / CardContent / CardFooter / CardTitle / CardDescription / CardAction

각 subcomponent는 native `<div>` props를 그대로 받습니다. `CardAction`은 `CardHeader` 안에서 우측 상단 영역으로 배치됩니다.

## 접근성

- Card 자체는 의미 요소가 아니므로 콘텐츠 의미에 맞춰 내부에 heading, paragraph, list, link, button을 사용합니다.
- 반복 카드 목록에서는 각 카드의 주요 제목을 heading 또는 링크로 제공합니다.
- 전체 카드가 클릭 가능해야 할 때도 내부 버튼과 링크의 focus 순서를 명확히 유지합니다.
