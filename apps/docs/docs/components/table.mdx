# Table

행과 열로 구조화된 데이터를 표시하는 컴포넌트 세트입니다.
`Table`은 overflow container를 함께 렌더하므로 좁은 화면에서는 표 내부에서 가로 스크롤됩니다.

  <DefaultExample />

## 설치

```bash
pnpm add @tcground/ui
```

```ts
import '@tcground/ui/theme.css';
```

## 사용법

```tsx
import { Table, TableBody, TableCell, TableHead, TableHeader, TableRow } from '@tcground/ui';

export function Example() {
  return (
    <Table>
      <TableHeader>
        <TableRow>
          <TableHead>카드</TableHead>
          <TableHead>평균가</TableHead>
        </TableRow>
      </TableHeader>
      <TableBody>
        <TableRow>
          <TableCell>리자몽 ex 151 SAR</TableCell>
          <TableCell>420,000원</TableCell>
        </TableRow>
      </TableBody>
    </Table>
  );
}
```

## 접근성

- 표 데이터에는 `TableHead`를 사용해 열 제목을 제공합니다.
- 제목이 필요한 표에는 `TableCaption`을 추가합니다.
- 숫자나 가격 열은 시각적 정렬만 바꾸고, 실제 텍스트는 읽기 쉬운 형식을 유지합니다.
- 행 클릭 동작을 추가할 때는 행 전체에만 의존하지 말고 명확한 button 또는 link를 제공합니다.
