# Label

form control의 이름을 표시하고 클릭 영역을 연결하는 라벨 컴포넌트입니다.
`@tcground/headless` Label primitive 기반이라 `htmlFor`로 입력 요소와 연결하면 클릭 시 해당 control로 focus가 이동합니다.

<ComponentPreview code={`<Label>이메일</Label>`}>
  <DefaultExample />
</ComponentPreview>

## 설치

```bash
pnpm add @tcground/ui
```

```ts
import '@tcground/ui/theme.css';
```

## 사용법

```tsx
import { Input, Label } from '@tcground/ui';

export function Example() {
  return (
    <div>
      <Label htmlFor='email'>이메일</Label>
      <Input id='email' type='email' />
    </div>
  );
}
```

### 필수 입력

필수 여부는 텍스트나 별도 설명으로 명시하고, 실제 검증은 input의 `required`와 서버 검증에서 처리합니다.

## 접근성

- form control에는 보이는 라벨을 연결합니다.
- 클릭 가능한 checkbox/radio 라벨은 control과 텍스트를 같은 `Label` 안에 둘 수 있습니다.
- required 표시만으로 검증을 끝내지 말고, input의 `required`와 서버 검증을 함께 사용합니다.
- disabled control과 연결된 라벨은 muted 스타일로 표현합니다.
