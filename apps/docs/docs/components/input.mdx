# Input

짧은 텍스트, 이메일, 비밀번호, 숫자, 검색어처럼 한 줄 값을 입력받는 기본 form control입니다.
`Input`은 native `<input>` props를 그대로 받으며, TCGround token 기반 border, focus ring, disabled, invalid 상태 스타일을 포함합니다.

  <DefaultExample />
</ComponentPreview>

## 설치

```bash
pnpm add @tcground/ui
```

```ts
import '@tcground/ui/theme.css';
```

## 사용법

```tsx
import { Input, Label } from '@tcground/ui';

export function Example() {
  return (
    <div>
      <Label htmlFor='card-name'>카드 이름</Label>
      <Input id='card-name' placeholder='Charizard ex' />
    </div>
  );
}
```

입력값 설명은 `Label`과 `htmlFor`/`id`로 연결합니다. 에러 상태는 `aria-invalid`를 사용하고, 에러 문구는 별도 텍스트로 제공합니다.

## 접근성

- `Label`의 `htmlFor`와 `Input`의 `id`를 연결합니다.
- 검증 실패는 `aria-invalid`로 표시하고, 에러 문구를 입력창 가까이에 배치합니다.
- placeholder는 라벨을 대체하지 않습니다.
- `disabled`는 입력, focus, form 제출 값을 모두 비활성화합니다.
