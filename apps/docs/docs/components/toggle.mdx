# Toggle

on/off 두 상태를 가진 단일 button 컴포넌트입니다. Switch 가 아닌 "툴바 버튼" (예: 굵게, 기울임) 형태의 토글 동작에 적합합니다. 상태는 `aria-pressed` 로 노출되어 보조 기기가 인식합니다.

<ComponentPreview
  code={`<Toggle>굵게</Toggle>`}
>
  <DefaultExample />
</ComponentPreview>

## 설치

```bash
pnpm add @tcground/headless-ui
```

```ts
import '@tcground/headless-ui/theme.css';
```

## 사용법

기본은 uncontrolled 입니다. 상태를 부모가 직접 관리하려면 `pressed` + `onPressedChange` 를 함께 사용합니다.

```tsx
import { Toggle } from '@tcground/headless-ui';

export function Example() {
  return <Toggle defaultPressed>굵게</Toggle>;
}
```

## 예제

### 기본 + 초기값

`defaultPressed` 로 초기 on 상태를 지정합니다. 컴포넌트가 내부적으로 상태를 관리합니다.

### Controlled

상위 상태와 동기화하려면 `pressed` 와 `onPressedChange` 를 함께 사용합니다.

<ComponentPreview
  code={`const [pressed, setPressed] = useState(false);

### Disabled

native `<button>` 의 `disabled` 속성을 그대로 사용합니다. 클릭과 키보드 활성화가 모두 차단됩니다.

## 접근성

- 상태는 `aria-pressed="true|false"` 로 노출됩니다.
- native `<button>` 이라 `Space` / `Enter` 키로 활성화되고, Tab 키로 focus 됩니다.
- controlled 사용 시 `pressed` 와 `onPressedChange` 를 짝지어 사용해야 보조 기기와 화면 상태가 어긋나지 않습니다.
- 단순 on/off (예: 알림 활성/비활성) 의미가 더 강하다면 Toggle 대신 native `<input type="checkbox" role="switch">` 또는 별도 Switch 컴포넌트를 고려하세요.
