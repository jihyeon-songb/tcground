# Input Group

입력창 양쪽에 아이콘, 단위 텍스트, action button을 붙여 하나의 form control처럼 보이게 하는 컴포넌트 묶음입니다.
`InputGroup`은 focus ring과 invalid 상태를 group 단위로 표시하고, 내부 control은 `InputGroupInput` 또는 `InputGroupTextarea`를 사용합니다.

## 사용법

```tsx
import { InputGroup, InputGroupAddon, InputGroupInput } from '@tcground/ui';

export function Example() {
  return (
    <InputGroup>
      <InputGroupAddon>⌕</InputGroupAddon>
      <InputGroupInput placeholder='카드 검색' />
    </InputGroup>
  );
}
```

## API Reference

### InputGroup

`InputGroup`은 native `<div>` props를 받으며 `role="group"`을 적용합니다.

### InputGroupButton

`Button` props를 기반으로 하며, 기본 `type`은 `button`, 기본 `variant`는 `ghost`, 기본 `size`는 `xs`입니다.

### InputGroupInput / InputGroupTextarea / InputGroupText

- `InputGroupInput`은 `Input` props를 기반으로 합니다.
- `InputGroupTextarea`는 `Textarea` props를 기반으로 합니다.
- `InputGroupText`는 단위, prefix, suffix 같은 보조 텍스트를 렌더합니다.

## 접근성

- group 전체가 하나의 control처럼 보이더라도 실제 입력 이름은 내부 input/textarea에 라벨로 제공합니다.
- icon만 있는 addon은 장식이면 `aria-hidden`, 의미가 있으면 보조 텍스트나 `aria-label`을 제공합니다.
- action button은 `InputGroupButton`을 사용하고, form 제출이 필요할 때만 `type="submit"`을 명시합니다.
- invalid 상태는 내부 control에 `aria-invalid`를 적용해 group border/focus ring과 보조 기기 상태를 함께 맞춥니다.
