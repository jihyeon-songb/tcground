# Separator

관련 콘텐츠 사이의 시각적 구획을 만드는 분리선입니다.
기본은 장식용 `horizontal` separator이며, 필요하면 `orientation="vertical"`로 수직 분리선을 만들 수 있습니다.

<ComponentPreview code={`<Separator />`}>
  <HorizontalExample />
</ComponentPreview>

## 설치

```bash
pnpm add @tcground/ui
```

```ts
import '@tcground/ui/theme.css';
```

## 사용법

```tsx
import { Separator } from '@tcground/ui';

export function Example() {
  return (
    <div>
      <div>최근 본 카드</div>
      <Separator />
      <div>관심 카드 목록</div>
    </div>
  );
}
```

## 접근성

- 단순 장식용 구분선은 기본값인 `decorative={true}`를 유지합니다.
- 콘텐츠 구조를 보조 기기에 전달해야 하는 구분선이면 `decorative={false}`를 사용합니다.
- 수직 separator는 충분한 부모 높이와 인접 항목 간격을 함께 설정합니다.
