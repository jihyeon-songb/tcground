# Sheet

페이지 맥락을 유지하면서 필터, 모바일 탐색, 짧은 설정을 표시하는 edge panel입니다.
`@tcground/headless` Dialog primitive를 기반으로 하므로 modal focus 관리와 ESC 닫기 동작을 제공합니다.

## 설치

```bash
pnpm add @tcground/ui
```

```ts
import '@tcground/ui/theme.css';
```

## 사용법

```tsx
import {
  Button,
  Sheet,
  SheetContent,
  SheetDescription,
  SheetFooter,
  SheetHeader,
  SheetTitle,
  SheetTrigger,
} from '@tcground/ui';

export function Example() {
  return (
    <Sheet>
      <SheetTrigger asChild>
        <Button variant='outline'>필터</Button>
      </SheetTrigger>
      <SheetContent side='right'>
        <SheetHeader>
          <SheetTitle>필터</SheetTitle>
          <SheetDescription>결과를 좁힙니다.</SheetDescription>
        </SheetHeader>
        <SheetFooter>
          <Button>적용</Button>
        </SheetFooter>
      </SheetContent>
    </Sheet>
  );
}
```

### SheetTrigger / SheetClose / Header / Footer / Title / Description

대응되는 `@tcground/headless` Sheet(Dialog) part 또는 native element props를 그대로 받습니다. `SheetTrigger`와 `SheetClose`는 `asChild`를 지원합니다.

## 접근성

- Content는 modal dialog semantics를 가지며, 열린 동안 focus가 Sheet 안에 머뭅니다.
- Title과 Description으로 panel 목적을 설명합니다.
- ESC 키, overlay 클릭, close button으로 닫을 수 있습니다.
- 모바일 navigation처럼 반복 사용되는 Sheet는 trigger와 첫 focus 대상의 위치를 예측 가능하게 유지합니다.
